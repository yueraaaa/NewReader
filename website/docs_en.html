📥 Installation

macOS

  1. Go to GitHub Releases and download the latest NewReader.dmg
  2. Open the DMG and drag NewReader.app into /Applications
  3. On first launch, if you see "unable to verify developer", go to System Settings → Privacy & Security and click "Open Anyway"

iOS

Build and install via Xcode. Open Package.swift, select the NewReaderiOS target, connect your iPhone, and run.

📡 Adding Feeds

  1. Press ⌘N or click the + button in the toolbar
  2. Enter an RSS / Atom / JSON Feed URL
  3. Click "Subscribe" — the feed title and articles will appear in the sidebar

OPML import supported: sidebar ⋯ → Import OPML, select a .opml file.

📁 Feed Management

  • Refresh: Right-click a feed → Refresh, or use the toolbar refresh button
  • Rename: Right-click a feed → Rename
  • Mark All Read: Right-click a feed → Mark all as read
  • Delete: Right-click a feed → Delete
  • Categories: Create categories to organize feeds; collapsible, renamable, deletable

🤖 AI Summarization & Translation

Supports OpenAI-compatible APIs (OpenAI, DeepSeek, Qwen, etc.).

  1. Open Settings (⌘,) → AI tab
  2. Select a model (DeepSeek-V3 / DeepSeek-R1)
  3. In articles: toolbar ✨ to summarize, 🌐 to translate
AI summarization and translation are provided by the NewReader backend — no need to configure API keys yourself.

📄 Full-text Extraction

Some feeds only provide summaries. Click the toolbar 📄 button to auto-fetch the full article from the original link.

💾 Offline Cache

Articles are automatically cached locally — read even without internet. Settings → Storage to view cache size and clear with one click.

🔐 Security Design

  • API keys encrypted in system Keychain — never stored in plaintext
  • WKWebView with JavaScript disabled + CSP policy to block script injection
  • URLs restricted to http/https only, private IP / localhost blocked (SSRF prevention)
  • HTML sanitization: strips script, iframe, and event handlers
  • OPML import capped at 5 MB
  • macOS App Sandbox with process-level isolation
