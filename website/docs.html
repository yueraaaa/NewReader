NEWREADER

使用文档

从安装配置到效率技巧，快速上手 NewReader 的全部功能。

安装 添加订阅源 订阅源管理 AI 摘要与翻译 语音朗读 全文提取 iCloud 同步 离线缓存 快捷键 安全设计

📥 安装

macOS

  1. 前往 GitHub Releases 下载最新 NewReader.dmg
  2. 打开 DMG，将 NewReader.app 拖入 /Applications
  3. 首次运行提示「无法验证开发者」，在 系统设置 → 隐私与安全性 中点击「仍要打开」

iOS

通过 Xcode 编译安装到设备。打开 Package.swift，选择 NewReaderiOS target，连接 iPhone 后运行。

📡 添加订阅源

  1. 按 ⌘N 或点工具栏 + 按钮
  2. 输入 RSS / Atom / JSON Feed 地址
  3. 点击「订阅」，Feed 标题和文章自动出现在侧栏

支持导入 OPML 文件：侧栏 ⋯ → 导入 OPML，选择 .opml 文件。

📁 订阅源管理

  • 刷新：右键 Feed → 刷新，或工具栏刷新按钮刷新全部
  • 重命名：右键 Feed → 重命名，输入新名称即可
  • 标为已读：右键 Feed → 标为已读
  • 删除：右键 Feed → 删除
  • 分类管理：新建分类收纳订阅源，可折叠、重命名、删除

🤖 AI 摘要与翻译

支持 OpenAI 兼容 API（适用于 OpenAI、DeepSeek、通义千问等）。

  1. 打开设置（⌘,）→ AI 标签
  2. 选择模型（DeepSeek-V3 / DeepSeek-R1）
  3. 在文章中使用：工具栏 ✨ 生成摘要，🌐 翻译内容
AI 摘要和翻译由 NewReader 后端统一提供，无需自行配置 API Key。

🔊 语音朗读 (TTS)

引擎说明网络要求
Apple 系统语音macOS / iOS 内置引擎，支持中文普通话、英日韩离线可用
自定义 TTS API接入任意 TTS 服务商，音质自然流畅需要网络

阅读文章时点工具栏 🔊 即可朗读，支持断点续播。设置 → 语音 标签可切换引擎并配置自定义 Endpoint 和 API Key。

📄 全文提取

部分 RSS Feed 只提供摘要。点工具栏 📄 按钮，自动从原文链接拉取完整正文。

☁️ iCloud 跨设备同步

同一 Apple ID 设备间自动同步订阅源、文章和阅读状态。

同步内容说明
订阅源及分类列表结构自动同步
阅读状态已读 / 未读状态同步
偏好设置主题、语言等偏好同步
API Key出于安全考虑不同步

💾 离线缓存

文章内容自动本地缓存，断网也能阅读。设置 → 存储 查看缓存大小并一键清除。

⌨️ 键盘快捷键

⌘N添加订阅
⌘,打开设置
⌘R刷新订阅源
⌘F搜索文章
J / ↓下一篇文章
K / ↑上一篇文章
Space向下滚动
⌘W阅读工作台

🔐 安全设计

  • API Key 使用系统 Keychain 加密存储，不落盘明文
  • WKWebView 禁用 JavaScript + CSP 策略，阻止脚本注入
  • URL 仅允许 http/https 协议，拦截私有 IP / localhost 防 SSRF
  • HTML 消毒移除 script / iframe / 事件处理器
  • OPML 导入文件大小上限 5MB
  • macOS App Sandbox 进程级隔离
