NewReader

精致、安全的 RSS 阅读器，内置 AI 摘要与翻译、TTS 语音朗读、全文提取，支持 macOS 与 iOS。

GitHub 下载 了解更多

多格式订阅

支持 RSS 2.0、Atom、JSON Feed 三种主流格式，自动发现并解析 Feed 内容。

AI 摘要与翻译

接入 OpenAI 兼容 API，一键生成文章摘要，支持中英日韩多语翻译。

TTS 语音朗读

利用系统语音引擎朗读文章内容，可调语速，支持中文普通话、英语、日语、韩语。

离线缓存

文章内容可在本地缓存，断网也能随时阅读，自动管理缓存空间。

全文提取

从摘要 Feed 自动提取完整文章正文，告别「点击阅读全文」的割裂体验。

安全优先

API Key Keychain 加密存储、WKWebView 禁用 JavaScript、HTML 消毒、URL 白名单防护。

支持平台

macOS 15+ iOS 18+ Apple Silicon

技术栈

Swift 6 SwiftUI SwiftData FeedKit WKWebView AVFoundation OpenAI API
